## Ads Report Dashboard
------------------------
A comprehensive and interactive Ads Performance Dashboard built using Power BI.
This project analyzes key advertising metrics such as impressions, clicks, conversions, cost analysis, and ROI to help marketers and decision-makers monitor campaign performance effectively.

# üìä Project Overview
-----------------------
The Ads Report Dashboard provides a data-driven view of marketing performance across different channels and campaigns.
It allows users to:

1. Track daily, weekly & monthly ad performance
2. Compare campaign-level KPIs
3. Analyze Cost vs Revenue trends
4. Monitor CPC, CPM, CTR, and Conversion performance
5. Identify top-performing ads & audiences
6. Detect spending anomalies and wastage

This dashboard helps stakeholders quickly understand where budget is being utilized and how effectively ads are performing.
